Spit on Shapes

Idle Brick Bomber

Idle Brick Bomber is the Android game that I'm currently developing in Unity2D. The game is in Early Access and I plan to add a few new features before full release.

Click here to try Idle Brick Bomber from the Google Play Store

As the name of the game indicates, it's an Idle game so you don't have to do (that) much. Like the classic Brick Breaker game, there will be bricks that can be destroyed by some sort of projectile. In this game there will be a cannon automatically shooting at bricks, and each destroyed brick will grant you some in-game currency that you can use to upgrade your cannon.

Some of the Idle elements that I added are:
Upon pausing or closing the game for more than 5 minutes, you'll be greeted with a prompt to claim some cash.
As mentioned, the cannon shoots automatically so you can progress without doing anything, but the further you get into the game, the longer it will take for the cannon to destroy all the bricks.
